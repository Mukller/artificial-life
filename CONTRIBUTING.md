# Contributing

## Как помочь

Принимаю PR с:
- Новыми типами организмов (хищники, паразиты, симбиоты)
- Улучшением визуализации (цвет по геному, тепловые карты)
- Оптимизацией (сейчас тормозит при >5000 организмов)
- Исправлением багов

## Как делать

```bash
git clone https://github.com/Mukller/artificial-life
cd artificial-life
pip install -e ".[dev]"
```

Пишите тесты если добавляете новую механику. Логику симуляции — в `life.py`.
Если добавляете параметр в геном — обновите раздел Genome в README.

## Стиль

- Типизация через `typing` (Python 3.10+)
- Форматирование: `ruff format`
- Без глобальных переменных, без side-effect при импорте
